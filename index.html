Let & Const Exercise-

Quiz:

Questions

    What is the difference between var and let?
    What is the difference between var and const?
    What is the difference between let and const?
    What is hoisting?

Answers

    When you are declaring a variable using the “var keyword” the value can be changed by way of re-declaring and re-assigning. However, when you're declaring a variable using the “let keyword” the value can be changed by way of re-assigning, but unlike the “var keyword” it can't be redeclared.
    Once again if you're declaring a variable using the “var keyword” the value can be changed by way of re-declaring and re-assigning. But when using the “const keyword” and you're declaring a variable, unlike “var and let”, after the original assignment of the value for “const”, you can't reassign or redeclare it.
    Just for clarification when you're declaring a variable using the “let keyword” the value can be changed by way of re-assigning but not re-decloration. When you're declaring a variable using "const" Once you've assigned a value to it, you'll be unable to reassign or redeclare it.
    In JavaScript, hoisting means that variable and function declarations are moved to the top of their scope during code execution. This allows you to use them before they are formally written in your code.
